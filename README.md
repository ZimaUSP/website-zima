# Website ZIMA

Este repositório contém o código-fonte do **site oficial da ZIMA – Grupo de Soluções Médico-Hospitalares da Poli USP**.
O projeto está atualmente em desenvolvimento e utiliza apenas tecnologias web básicas:

* **HTML**
* **CSS**
* **JavaScript**

Não há frameworks ou sistemas complexos - o objetivo é manter um código acessível, simples de manter e fácil de hospedar.

---

## Como Executar o Site Localmente

Existem várias formas de testar o site em ambiente local:

---

### 1. Abrindo diretamente no navegador

Basta clonar o repositório e abrir o arquivo `index.html` no seu navegador.

---

### 2. Usando o Python (servidor HTTP simples)

Se você tiver o **Python 3** instalado, execute no diretório do projeto:

```bash
python3 -m http.server
```

Depois, acesse no navegador:

```
http://localhost:8000
```

---

### 3. Usando o VS Code (Live Server)

Se você utiliza o **VS Code**, pode instalar a extensão **Live Server** e iniciar o servidor clicando em:

> **Go Live** no canto inferior direito

Isso permite recarregamento automático a cada alteração.

---

## Objetivo do Site

O site tem como finalidade:

* Apresentar a ZIMA, suas áreas e projetos
* Divulgar o trabalho desenvolvido em Saúde e Tecnologia
* Facilitar o contato com novos membros e parceiros
* Servir como vitrine institucional do grupo
